Evaluator Instructions


Perform Evaluation

  1. Select 'Current Assesments'
  2. Locate the list of assigned evaluations
  3. Select the evaluation to perform
  4. The actions required to perform an evaluation depend on the evaluation tool

  1. Access the rubric evaluation
      2. Check out 'Access Evaluation'
  2. Select 'Start Evaluations'
  3. Select the students name to evaluate from the dropdown list
  4. Select a score for each score field
  5. Select 'Evaluate'

  1. Access the evaluation the scores will be added to
    • Check out 'Access Evaluation'
  2. Select 'Upload Test Score'
  3. Fill in the Full Marks field with the total number of points the exam is worth
  4. Select 'Choose File'
    • Make sure the file follows the correct format
    • The formatting example can be found by selecting 'download example file'
  5. Once the file has been located, select 'Upload'

  1. Access the evaluation the scores will be added to
    • Check out 'Access Evaluation'
  2. Select 'Add test scores'
  3. Select the student's name from the dropdown box
  4. Fill in the marks obtained field with the number of points the student received
  5. Select 'Add Score'



Other

  1. Select 'Past Assesments'
  2. Locate the title of the evaluation and the name of the student whose evaluations needs editing
  3. Select 'Edit'
  4. Perform edits
  5. Select 'Save'

The evaluator's progress is represented by a percent circle that displays the percent of completed evaluations for the measure that has been selected
  1. Access the desired evaluation
    • Check out 'Access Evaluation'
  2. The progress circle can then be viewed

The evaluator can receive messages from admins of their program, and other programs they are evaluators for
  1. Access the rubric evaluation
      2. Check out 'Access Evaluation'
  2. Select 'Start Evaluations'
  3. Review the rubric
  4. Select 'Current Assesments' to exit the rubric

  1. Select the message icon on the navigation bar
