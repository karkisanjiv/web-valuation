Coordinators

Coordinators are the designated person in charge of each program's assesments. They create the outcomes, measures, and rubrics that make up the assesment process, and invite evaluators to perform the assesments.

  1. Select 'Invite Coordinators'
  2. Enter the email for the potential coordinator
  3. Input the department name
  4. Select 'Invite'
    • The coordinator will receive an email containing a link to the login page
    • They must register with the same email address used to invite them

  1. Select the message icon on the navigation bar
  2. Select the coordinator to message from the list of coordinator names
  3. Type the message in the message box
  4. Select 'Send'
