Evaluator Help

To Start Evaluation
  1. Select an evaluation from the list of evaluations to complete
    • The list of students requiring an evaluation will be displayed
  2. Select 'Start Evaluations' to get started!
To Perform Rubric Evaluation
  1. Select student's name from the dropdown box at the top
  2. Select the score for each criteria from the dropdoen boxes
  3. Select 'Evaluate' once all of the score fields have been completed
To Edit A Previous Rubric
  1. Select 'Past Assesments' from the dashboard menu
  2. Select 'Edit' by the rubric needing to be edited
  3. Perform edits
  4. Select not sure yet
